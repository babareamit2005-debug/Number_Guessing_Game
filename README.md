## Description
The system generates a random number between 1 and 100.
The user guesses the number and receives hints until the correct number is guessed.

## Features
- Random number generation
- User input using Scanner
- Loop until correct guess
- Simple console-based game

## How to Run
1. Install JDK
2. Compile: javac NumberGuessingGame.java
3. Run: java NumberGuessingGame
